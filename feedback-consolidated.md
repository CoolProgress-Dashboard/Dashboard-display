### MEPS Benchmarking — AC Units (Data Errors & Methodology Issues)
- **Source: Patrick (HEAT, 2026-05-04)** — Multiple benchmarking issues identified for room ACs. Clarification on calculation methodology required before correcting:
  - **SADC and EAC MEPS too low** — values shown do not align with U4E model regulations. Must be corrected.
  - **U4E capacity range mismatch** — U4E MEPS uses 4.5–9.5 kW range while country values appear to use 2.5–5 kW. This makes U4E reference look lower than it actually is. For example, China and Singapore appear higher than U4E as a result, which is misleading. Note: updated MRG capacity ranges are now <6 kW, 6–14 kW, and 14–70 kW.
  - **U4E high-efficiency label value wrong** — shown as 6.5 ISO CSPF, but for the capacity range used it should be 7.6, and for the lower (more comparable to country) range it should be 8.0. Also note: the MRG presents this as a high-efficiency reference, not a recommended threshold for the highest label class.
  - **Nigerian MEPS too low** — should align with U4E levels by 2031. The conversion methodology is not explained — the graph only cites U4E as the source without showing how the conversion was done.
  - **Saudi Arabia MEPS too low** — should also be higher (Patrick's initial assessment).
  - **EER/CSPF conversion for variable-speed units** — for countries using EER or similar metrics (Australia, Saudi Arabia), conversion to CSPF only works for fixed-speed units. Variable-speed units convert differently. Options: (A) show separate MEPS values for fixed-speed and variable-speed products, or (B) clarify in methodology that only the fixed-speed equivalent is shown.
  - **Unverified future levels** — India 2033 and China 2025 levels shown in the graph as "adopted — future effective date" but Patrick cannot find officially published levels for these. Requires verification (flagged to Sai for the Indian case).
  - **Optional test methods (China, Brazil)** — some countries allow optional test methods which produce different (usually higher) efficiency values, making comparisons difficult. At minimum, clarify that comparisons are based on ISO CSPF using mandatory tests only.
  - **Japan Top Runner** — Japan uses the Top Runner approach, which is fundamentally different from a typical MEPS framework. How MEPS is being represented in the comparison for Japan needs to be clearly explained in the methodology.

### MEPS Benchmarking — Domestic Refrigerators (Data Errors & Methodology Issues)

  - **EU EEI not explicitly named** — the methodology states "all are normalized to a common Energy Efficiency Index (EEI)" but does not specify which EEI. ### Wet-Bulb Heat Stress / WBGT
- **Source: Session 2 (2026-04-22)** — Wet-bulb temperature / WBGT (Wet Bulb Globe Temperature) is a critical but under-represented dimension. Highest-risk regions: India, Pakistan, Bangladesh, Persian Gulf, SE Asia, eastern China, coastal Brazil.
